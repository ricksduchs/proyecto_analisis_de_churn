
# Análisis de Churn de Clientes - Fit.ly

## Descripción general
Este proyecto analiza el churn de clientes para una aplicación de fitness con el objetivo de identificar los principales factores de abandono de usuarios.

## Principales hallazgos
- Los usuarios gratuitos presentan tasas de churn significativamente más altas  
- El nivel de engagement del usuario impacta fuertemente la retención  
- Los tickets de soporte tienen una influencia mínima en el churn  

## Herramientas
Python, Pandas, Matplotlib, Seaborn

## Recomendaciones
Dado que la inactividad es el principal impulsor del churn  
La empresa debe priorizar el aumento del compromiso con el producto, asegurando que los usuarios perciban valor de manera constante.  

Dado que los usuarios que abandonan tienen mayores tiempos de resolución  
Es importante reducir la fricción en la experiencia de soporte mejorando la eficiencia en la resolución de problemas.  

Dado que el churn es significativamente mayor entre los usuarios gratuitos  
La empresa debe reevaluar la propuesta de valor del plan gratuito e implementar estrategias para impulsar la conversión a planes de pago.
